Firm LCOE Calculator   Experimental

This tool estimates the cost of delivering firm, round-the-clock renewable electricity from a co-located solar, wind, and battery hybrid system. Based on IRENA (2026), 24/7 renewables: The economics of firm solar and wind (opens in new tab).

The app is experimental and under testing. If you would like to try it, please fill out this short form (opens in new tab) to request an API key. You will also need a Renewables.ninja (opens in new tab) key.

Tested it? Share your feedback (opens in new tab).

  1. Select a site by clicking on the map
  2. Define the existing system - installed solar, wind, and/or BESS capacities
  3. Select which technologies are allowed to provide firming
  4. Set the reliability target
  5. Click Simulate
