# Firm LCOE Calculator

**Round-the-Clock (RTC) Solar, Wind, and Battery Hybrids**

An interactive web tool for estimating the cost of delivering firm, round-the-clock renewable electricity from a co-located solar, wind, and battery hybrid system.

Based on [IRENA (2026), 24/7 Renewables: The Economics of Firm Solar and Wind](https://www.irena.org/Publications/2026/May/24-7-renewables-The-economics-of-firm-solar-and-wind).

🔗 **Live tool:** [iitc-te.github.io/firm-lcoe-calculator](https://iitc-te.github.io/firm-lcoe-calculator/)
